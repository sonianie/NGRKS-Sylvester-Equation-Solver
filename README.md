# NGRKS Sylvester Equation Solver

This repository contains MATLAB implementations and numerical experiments for the **Novel Global Rational Krylov Subspace (NGRKS)** method for solving large-scale Sylvester matrix equations of the form:

**AX + XB + FG' = 0**

The implementation is based on the research paper:  
*S. Nie and H. Dai, "A new iterative method for solving large-scale Sylvester matrix equations," Journal of Computational Mathematics, submitted.*

## 📁 File Structure

- **Core Solver**
  - `NGRKSn_sylv.m`: Main implementation of the NGRKS algorithm for solving Sylvester equations.

- **Error & Residual Analysis**
  - `err_NGRKSn_sylv.m`: Computes and compares true error norms with theoretical error bounds.
  - `res_NGRKSn_sylv.m`: Computes and compares true residual norms with theoretical residual bounds.

- **Numerical Examples**
  - `Example1_ERR.m` / `Example1_RES.m`: Experiments for error/residual norm and matrices from [The University of Florida Sparse Matrix Collection](https://sparse.tamu.edu) (Example 6.1 from the paper [1]).
  - `Example2_C.m`: Experiment for different cycle parameter (c = 1 to 9) and matrices generated by `fdm_2d_matrix.m`,  (Example 6.2 from the paper [1]).
  - `Example3_case1_E1.m` / `Example3_case1_E2.m`: Comparison with other methods using large-scale matrices from Examples 1 and 2 of [2] (Case 1).
  - `Example3_case2.m`: Additional comparison using matrices from Example 6.3 of [1] (Case 2)


## 🚀 Features

- **Adaptive Pole Selection**: Automatically chooses optimal poles within estimated eigenvalue ranges.
- **Flexible Cycle Parameter**: Supports different cycle parameters (`c=1,2,...,9`) for the restarted Krylov method.
- **Efficient Low-Rank Approximation**: Returns solutions in factored form `X ≈ Z1 * Z2'`.
- **Comprehensive Analysis Tools**: Includes functions to verify theoretical error and residual bounds.

## 📊 Usage Examples

The provided example scripts demonstrate how to:
1. Set up problem matrices from PDE discretizations or sparse matrix collections
2. Configure solver parameters (tolerance, eigenvalue estimates)
3. Run the NGRKS solver with different cycle parameters
4. Analyze convergence behavior and computational performance


## 📝 Notes

- The code requires MATLAB with standard toolboxes (including sparse matrix support).
- For examples using Matrix Market files (e.g., `nos6.mtx`), download matrices from [The University of Florida Sparse Matrix Collection](https://sparse.tamu.edu).
- All scripts include detailed comments explaining each computational step.
